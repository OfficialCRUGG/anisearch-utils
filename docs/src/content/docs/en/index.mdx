## What is CRUGG's aniSearch Utils?

CRUGG's aniSearch Utils is a browser extension that adds <abbr title="A user of a piece of software that tries to make the most efficient use of the software they're using.">power user</abbr> tweaks to aniSearch.

These features can be individually toggled and are intended for users who actively contribute to the database and want to make their workflow more efficient.
