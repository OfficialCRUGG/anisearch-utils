import { Aside } from '@astrojs/starlight/components';

- **Kategorie:** Anime Seite
- **Modul-ID:** `linkMau2`

Dieses Modul fügt auf Anime-Seiten einen Button hinzu, über den man den japanischen Titel des jeweiligen Anime auf Mau2 suchen kann. Mau2 ist eine japanische Webseite, die Listen von Synchronsprechern für Anime bereitstellt.

<Aside type="tip">
Da dieser Button wie ein normaler Link funktioniert, kannst du die mittlere Maustaste (das Scrollrad drücken) verwenden, um den Link in einem neuen Tab zu öffnen.
</Aside>
