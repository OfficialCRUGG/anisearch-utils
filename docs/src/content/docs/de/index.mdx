## Was ist CRUGG's aniSearch Helfer?

CRUGG's aniSearch Helfer ist eine Browsererweiterung, die einige Anpassungen für <abbr name="Jemand, der versucht, Software zu effizient wie möglich zu nutzen.">Power-User</abbr> auf aniSearch hinzufügt.

Diese Funktionen können alle individuell aktiviert und deaktiviert werden und richten sich an Nutzer:innen, die aktiv an der Datenbank mitwirken und ihren Workflow effizienter machen möchten.
