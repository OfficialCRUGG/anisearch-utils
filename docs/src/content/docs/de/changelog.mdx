## v1.0.2 [2024.12.28]

### Allgemein

- Die Versionnummer wird nun im Popup der Erweiterung auf dem "Informationen" Tab angezeigt

### Module

#### Mau2 Übersetzer

- Episodentabellen mit einer dritten Spalte für zusätzliche Icons werden nun korrekt identifiziert (Fixes [#2](https://github.com/OfficialCRUGG/anisearch-utils/issues/2))
- Episodentabellen mit japanischen Zahlen werden nun korrekt identifiziert (Fixes [#3](https://github.com/OfficialCRUGG/anisearch-utils/issues/3))

## v1.0.1 [2024.12.18]

Dieses Update beinhaltet hauptsächlich kleinere Änderungen, um den Firefox-Erweiterungsrichtlinien zu entsprechen.

### Allgemein

- Das 32x32 Icon hat nun die korrekte Größe
- `.innerHTML` wird nicht mehr verwendet
- Die Firefox Mindestversion wurde von 109 auf 126 erhöht

## v1.0.0 [2024.12.18]

### Allgemein

- Die Erweiterung wurde von "aniSearch Helfer" in "CRUGG's aniSearch Helfer" umbenannt, um mögliche Verwirrungen zu vermeiden.
- Unterstützung für Firefox hinzugefügt
- Komplett überarbeitete Benutzeroberfläche

### Module

#### Volle Breite

- Das Modul wurde entfernt, da eine ähnliche Funktionalität bereits auf aniSearch existiert.

#### Mau2 Übersetzer

- Ein Button wurde hinzugefügt, der es ermöglicht, alle separaten Episodentabellen in eine einzige Tabelle mit einem "Episoden"-Feld und ohne Duplikate zusammenzuführen.

#### Mau2 Link

- Der japanische Name wird nun korrekt extrahiert

#### Tastaturkürzel

- Dieses neue Modul wurde hinzugefügt, um das Hinzufügen vieler Einträge zu einer Liste (derzeit nur Episoden und Synchronsprecher) zu beschleunigen.

## v0.2.0 [2024.12.10]
